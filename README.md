# The Pit

An idle-roguelite delve game. Descend into a procedural pit, farm resources, buy passives, fight the occasional boss. OGame meets Melvor Idle meets Path of Exile's Delve — in a terminal UI.

> Status: scaffolding. Nothing playable yet. See `brainstorming/` for the direction.

## Stack

- **React 19** + **Vite 7** — SPA, no SSR (idle game lives client-side)
- **TanStack Router** (file-based) + **TanStack Query**
- **Zustand** — local UI state
- **PixiJS v8** — combat / delve map rendering
- **Motion** — UI animations
- **Convex** — backend, auth, server-authoritative game state
- **Tailwind v4** — styling
- **Monaspace / JetBrains Mono** — terminal-inspired typography
- **Vitest** + **Playwright** — tests

## Getting started

```bash
npm install
cp .env.example .env.local   # already done for this machine
npm run dev:all              # runs vite + convex dev in parallel
```

First time only — link the Convex deployment:

```bash
npx convex dev
# choose: existing project → "the-pit" (aware-goose-251)
```

## Scripts

| Command | What it does |
|---|---|
| `npm run dev` | Vite dev server (5173) |
| `npm run convex:dev` | Convex functions hot-reload |
| `npm run dev:all` | Both, concurrently |
| `npm run routes:gen` | Regenerate `src/routeTree.gen.ts` |
| `npm run build` | Typecheck + generate routes + bundle |
| `npm run typecheck` | TS type check only |
| `npm test` | Vitest unit tests |
| `npm run test:e2e` | Playwright e2e |

## Layout

```
src/
  routes/           TanStack Router file-based routes
  game/             Core game logic (pure, no React)
  components/       React components
  stores/           Zustand stores (UI state only)
  pixi/             PixiJS renderers
convex/
  schema.ts         DB schema
  _generated/       Auto-generated by convex dev
brainstorming/      Research, game loop drafts, design notes
```

## Design pillars

1. **Active Delve first** — offline = flavor, not core progression.
2. **Server-authoritative** — no `localStorage.setItem('gold', 1e9)`.
3. **Terminal UI** — subtle pixel/ASCII aesthetic, monospace-first.
4. **One boss for now** — depth over breadth.
